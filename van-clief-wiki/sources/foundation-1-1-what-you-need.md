# foundation-1-1-what-you-need

> Packaged inside `.wiki/raw/the-foundation.md`, lines ~1-69. Part of the Foundation course (Module 1, Lesson 1).

## Summary

The five-minute setup lesson. Van Clief lists three prerequisites: a Claude account, a code editor, and Node.js (only if you plan to use Claude Code). Plans are triaged: Free for learning, Pro ($20/mo) is where most members sit and gets Claude Code access, Max ($100/mo) for heavy daily use. The full install walkthrough with animation use-case framing is deferred to Foundation 4.1.

## Key claims

1. Three things are sufficient to start: Claude account, a code editor, Node.js.
2. Pro subscription ($20/mo) is the recommended default because it unlocks Claude Code.
3. VS Code and Cursor are the two recommended editors; Cursor is a VS Code fork with AI baked in.
4. Node.js is only required for Claude Code; skip it for browser-only workflows.
5. Paid plan is not required to get value from the course; an account is.
