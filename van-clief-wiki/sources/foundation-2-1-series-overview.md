# foundation-2-1-series-overview

> Packaged inside `.wiki/raw/the-foundation.md`, lines ~336-464. Foundation Module 2, Lesson 1.

## Summary

The map for the six-part Abstraction series. Each video takes one idea from computing history and uses it to explain something about AI that most people get wrong. The throughline: AI is not new magic, it is the next layer in a stack that has been building for 70 years (longer, actually). Every video has a written companion designed to be read alone, handed to Claude, or used as reference after watching. The lesson ends with a "start somewhere" matrix keyed to the reader's goal.

## Key claims

1. The six-part series tracks one throughline: AI is the newest layer in a 70-plus-year reliability-engineering stack.
2. Each lesson in the series has a written companion that stands on its own; the companions are not summaries.
3. Viewing order is flexible, but the last video ties everything together.
4. Start matrix: new viewers start with 2.2 in order; business/investment focus starts with 2.4 and 2.5; technical deep-cut starts with 2.7 and works backward.
5. The series is an abstraction-ladder tour: stack (2.2), memory template pattern (2.3), commoditization ladder (2.4), orchestration vs intelligence (2.5), production pipeline (2.6), full system deconstruction (2.7).
