## Summary

The roadmap. What the reader has at the end of Foundation: tools installed (1.1), a folder system (1.2), a prompting framework (1.3), six abstraction videos (Section 2), the full three-layer folder architecture (Section 3), and Claude Code running on the machine (Section 4). From here the paths are Level 2 (Implementation Playbooks - Building Animations, The Ultimate Browser), Level 3 (Building Your Stack - Custom UI for Claude, Remote Access for Claude), Premium/The Vault ($27/mo with Workspace Blueprint, folder guides, 60/30/10 course, community), or VIP/The Drawing Room ($97/mo with live Afternoon Tea sessions, bespoke workspace builds, direct Discord access). Level 2 unlocks through community engagement, not payment.

## Key claims

1. Foundation closes with a working personal orchestration system: tools, folder, prompt framework, abstraction knowledge, three-layer architecture, Claude Code.
2. Level 2 (Implementation Playbooks) is unlocked by community engagement; currently includes Building Animations and The Ultimate Browser.
3. Level 3 (Building Your Stack) currently includes Custom UI for Claude and Remote Access for Claude.
4. Premium/The Vault ($27/mo) contains the Workspace Blueprint, Production CLAUDE.md examples, folder guide, prompting one-pager, animation spec templates, workflow starter packs, the Claude Skills Manual, and the 60/30/10 Framework course.
5. VIP/The Drawing Room ($97/mo) is not a content library - it is live Afternoon Tea sessions every two weeks, bespoke folder builds by Van Clief, and smaller Discord channels.
6. Van Clief's outbound presence: YouTube (JEVanClief, creamer_ai), Instagram (lostandlucky, creamer.ai), X (VanCliefMedia, creamer_ai), TikTok (jakevanclief, creamer.ai), LinkedIn.
