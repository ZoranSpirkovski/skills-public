# van-clief-wiki

A navigable body of Jake Van Clief's teachings on AI-native folder workflows — the three-layer routing architecture, routing tables, naming conventions as lightweight DB, folder-as-workspace ergonomics, and the broader abstraction-layer / computational-orchestration arguments he cites as foundational.

Compiled by Zoran Spirkovski to support the `pocket-clief` plugin in this repo: when the skill is edited, the fidelity check consults this wiki.

## Structure

- `SCHEMA.md` — page-type rulebook (source / concept / entity / synthesis).
- `index.md` — full page inventory grouped by category.
- `log.md` — append-only ingest and lint log.
- `sources/` — one page per Van Clief lesson or document (summary + citations).
- `concepts/` — recurring ideas across sources.
- `entities/` — people, products, and organizations cited.
- `synthesis/` — cross-source pattern pages.

## Raw transcripts are not included

Some source pages reference `.wiki/raw/*` paths. Those refer to Van Clief's original course transcripts, which live in the compiler's private working copy and are not redistributed here. The summaries, key-claim lists, and cross-citations in this wiki are original synthesis attributable to public discussion of Van Clief's publicly-shared teachings.
