# uiux-pro-skill

A skill package (prompts, examples, guidelines) that teaches [[claude-code]] better design sense for front-end work. Not a repo to run but context Claude reads. Included in Van Clief's Building Your Stack repo tour both as a practical recommendation and as an example of how to create domain-specific skills.

## Claims

- Skill package: prompts and instructions Claude reads to produce better-designed front-end work [sources/stack-1-4-repo-tour-open-source-references]
- Worth studying for skill structure (prompts, examples, guidelines), encoded design principles, and how skills can shape Claude's output style [sources/stack-1-4-repo-tour-open-source-references]
- If you are building a front-end with Claude, this skill makes the output look better [sources/stack-1-4-repo-tour-open-source-references]
- Good example of how to create your own skills for specific domains [sources/stack-1-4-repo-tour-open-source-references]
