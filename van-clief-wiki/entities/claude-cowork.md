# claude-cowork

Anthropic's folder-aware AI interface - operates inside folders similarly to [[claude-code]] but without requiring terminal setup or a separate IDE. Mentioned in Foundation 3.1 as a suitable alternative for users who find [[vs-code]] overwhelming.

## Claims

- Operates inside folders "the same way" as the rest of the three-layer system, without requiring VS Code [sources/foundation-3-1-full-walkthrough]
- Positioned as a gentler entry point than [[claude-code]] / [[vs-code]] for non-developer users [sources/foundation-3-1-full-walkthrough]
- Level 2 on the [[tool-ladder]]: more control than chat but no full filesystem access [sources/stack-1-1-starting-the-build-process]

## Relationships

- Alternative runtime to [[claude-code]] for [[three-layer-routing]]
