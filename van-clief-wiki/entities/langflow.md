# langflow

A visual framework for building LLM applications: drag-and-drop interface for creating AI workflows. Included in Van Clief's Building Your Stack repo tour as a study reference for visualizing LLM chains and agents.

## Claims

- Visual framework for building LLM applications with a drag-and-drop interface [sources/stack-1-4-repo-tour-open-source-references]
- Worth studying for how it visualizes LLM chains and agents, the component system (each node is a component), and how it bridges visual representation to actual execution [sources/stack-1-4-repo-tour-open-source-references]
- Relevant to anyone building visual interfaces for AI workflows [sources/stack-1-4-repo-tour-open-source-references]
