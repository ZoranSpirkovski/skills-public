# astro

Static-site framework used as one of the possible tech-stack choices in the Playbooks Module 3 website build. When Astro (or React, or another build-based stack) is selected, GitHub Pages deploys via a GitHub Actions workflow that runs the build step. Claude Code generates the workflow file.

## Claims

- Named as a build-based option alongside React for the GitHub Pages build flow [sources/playbooks-3-2-github-and-folder-structure]
- Build-based deploys run GitHub Actions; the workflow file (`.github/workflows/deploy.yml`) is generated by Claude Code [sources/playbooks-3-2-github-and-folder-structure]
