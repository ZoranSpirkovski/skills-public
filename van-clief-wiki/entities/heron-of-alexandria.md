# heron-of-alexandria

Greek engineer, 1st century CE. Known for an automatic theatre (~62 CE, dated via a referenced lunar eclipse) that performed a 10-minute Dionysian myth using programmable ropes, knots, and falling weights. Van Clief cites him as the earliest documented example of programmable automation - same logical structure as modern conditional code (if-this-then-that), but encoded mechanically.

## Claims

- Built an automatic theatre around 62 CE using ropes, knots, and a millet-seed timer to program a sequence of fire, liquid, and movement effects [sources/archive-1-2-2000-year-overnight-success]
- The programming was binary-like: wrapped vs. unwrapped cord around rotating drums determined action sequencing [sources/archive-1-2-2000-year-overnight-success]
- Used millet-seed for timing because sand was too fast and water too unreliable in Egyptian heat [sources/archive-1-2-2000-year-overnight-success]
