# cursor-editor

AI-native code editor built as a fork of [[vs-code]]. Named alongside VS Code as one of the two recommended editors in the Foundation course. Free to start. Offers AI features baked in out of the box. Van Clief names it as a valid alternative but uses VS Code + Claude Code in his own demos. Windsurf, Copilot, and Antigravity are named as peers in the same VS Code-fork category.

## Claims

- Built on VS Code with AI features baked in; free to start [sources/foundation-1-1-what-you-need]
- Named alongside VS Code (and "antigravity") as a valid editor choice for Foundation setup [sources/foundation-1-1-what-you-need]
- Part of a broader family of VS Code forks including Windsurf, Copilot, and Antigravity, which support different models and workflows [sources/foundation-4-1-install-and-first-use]

## Relationships

- Fork of [[vs-code]]
- Alternative runtime host for [[claude-code]] (though the course demos use VS Code)
