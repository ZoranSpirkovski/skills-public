# science-museum-london

Museum in London cited in Archive 1.2 for two reconstructions: [[al-jazari]]'s scribe clock (built in 1976 from 800-year-old instructions - it worked perfectly) and [[charles-babbage]]'s Difference Engine No. 2 (built 1985–1991 using only Babbage-era techniques - it calculates perfectly).

## Claims

- 1976: reconstructed Al-Jazari's scribe clock from his manuscript; it worked [sources/archive-1-2-2000-year-overnight-success]
- 1985–1991: built Babbage's Difference Engine No. 2 using only period-accurate techniques; it calculates perfectly [sources/archive-1-2-2000-year-overnight-success]
