# open-webui

A self-hosted open-source web interface for LLMs. Supports multiple providers, local models, and has a clean chat UI. Included in Van Clief's Building Your Stack repo tour as a study reference for multi-provider architecture.

## Claims

- Self-hosted web interface for various LLMs; supports multiple providers and local models [sources/stack-1-4-repo-tour-open-source-references]
- Worth studying for multi-provider architecture: how they abstract different AI backends, conversation management, system prompts and presets [sources/stack-1-4-repo-tour-open-source-references]
- Useful if you are building something that needs to work with multiple AI providers [sources/stack-1-4-repo-tour-open-source-references]
