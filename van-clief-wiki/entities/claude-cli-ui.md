# claude-cli-ui

An open-source terminal UI (TUI) for [[claude-code]]. Stays in the terminal but adds visual organization beyond the raw CLI. Reference repo in Van Clief's Building Your Stack research list.

## Claims

- Terminal UI (TUI) that stays in the terminal but adds better visual organization [sources/stack-1-4-repo-tour-open-source-references]
- Worth studying for how it parses and displays Claude output, keyboard shortcuts, and the balance between terminal simplicity and added features [sources/stack-1-4-repo-tour-open-source-references]
- Good fit for users who want to stay in the terminal but want a better experience than raw CLI [sources/stack-1-4-repo-tour-open-source-references]
- Surveyed during Van Clief's initial research phase alongside [[claude-code-web]] and [[op-code]] [sources/stack-1-2-how-i-use-claude-code-in-build]
