# substack

Newsletter and publishing platform. Van Clief cites it as the working proof that writers can own their distribution and audience rather than surrender both to a publisher. Used in Lesson 3.2 as one of three precedent industries (alongside GitHub for code and Bandcamp for music) that prove the ownership model can work.
