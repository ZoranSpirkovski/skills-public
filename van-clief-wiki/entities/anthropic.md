# anthropic

AI safety company that builds Claude and the surrounding product suite (Claude Code, Claude Desktop, Claude for Chrome, Claude Design, Claude Cowork). Van Clief references Anthropic across the corpus primarily through its tools; the company's direct claims appear in embedded product announcements (for example, the Sonnet 4.5 announcement for the Chrome extension).

## Claims

- Publishes the Claude for Chrome extension, powered by Sonnet 4.5 [sources/playbooks-2-1-setting-up-claude-in-chrome]
- Publishes Claude Design, which wraps Claude Code plus skills behind an interface for design systems, decks, and animated content [sources/playbooks-1-5-claude-design-folder-structure]
- Maintains a public GitHub repository of skills, used as reference material in the Playbooks [sources/playbooks-1-5-claude-design-folder-structure]
