# devstral

Mistral's coding-specialized model. Available via Mistral's API for users with light local hardware, or runnable locally for full control. Named in Playbooks 1.5 as one of four fallbacks for running a Claude Design exported project without Claude.

## Claims

- Built specifically for software engineering tasks [sources/playbooks-1-5-claude-design-folder-structure]
- Accessible via API or local install [sources/playbooks-1-5-claude-design-folder-structure]
- Viable for running Claude Design exports outside Claude [sources/playbooks-1-5-claude-design-folder-structure]
