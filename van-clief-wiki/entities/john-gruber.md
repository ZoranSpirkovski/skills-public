# john-gruber

Creator of [[markdown]] in 2004. Named after a play on "markup language" - markdown strips the tag complexity of HTML and keeps plain English with a few formatting symbols. Credited by Van Clief as the source of the format that every file in the folder methodology uses.

## Claims

- Created markdown in 2004 [sources/folder-system-transcript] [sources/foundation-3-1-full-walkthrough]
- Designed it to be "readable as plain text that can also render into a formatted document" [sources/folder-system-transcript] [sources/foundation-3-1-full-walkthrough]
