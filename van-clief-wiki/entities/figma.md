# figma

Collaborative design tool. Referenced as an alternative source format in Playbooks 1.2 (any design tool that exports SVG works) and as a direct input to [[claude-design]] in Playbooks 1.5. Van Clief's position: "Figma is not useless anymore" in the AI era. The designer invests in the Figma file first; the downstream tools amplify the exact design.

## Claims

- Any design tool that exports SVG, including Figma, works as the starting point for the Illustrator-to-animation pipeline [sources/playbooks-1-2-illustrator-to-web-animations]
- Claude Design accepts Figma files directly as an input to the design-system generation step [sources/playbooks-1-5-claude-design-folder-structure]
- Van Clief's framing: spend time in Figma getting the first draft exactly right, then the AI system amplifies it [sources/playbooks-1-5-claude-design-folder-structure]
- "A designer will use Claude Design better than almost anyone else" [sources/playbooks-1-5-claude-design-folder-structure]

## Relationships

- Input to [[claude-design]]
- Alternative source to [[adobe-illustrator]] for [[svg-layer-naming]] workflows
